Assent javascript model validation

Assent is a validation rule definition and execution library for javascript models.

Install

bower install assent

Configuration

Lets use this simple model

function UserCreate() {
  this.username = null;
  this.password = null;
  this.passwordConfirmation = null;
  this.email = null;
}

Defining a model validator

var userValidator = new Validator(function (c) {
  c.ruleFor('username', function (f) {
    f.notEmpty()
      .withMessage('username is required');
  });

  c.ruleFor('password', function (f) {
    f.notEmpty()
      .withMessage('password is required');
  });

  c.ruleFor('passwordConfirmation', function (f) {
    f.notEmpty()
      .withMessage('password confirmation is required');

    f.matches(function (obj) { return obj.password; })
      .withMessage('password confirmation must match password');
  });
});

The above example uses the notEmpty and matches validator.

We stated that the username, password and passwordConfirmation fields are all required. Furthermore the value entered into passwordConfirmation must match the value entered into password.

Apply the validator to the UserCreate model

addressValidator.applyTo(UserCreate);

Make it so!

var user = new UserCreate();
user.username = 'octocat';
var result = user.$validate();
