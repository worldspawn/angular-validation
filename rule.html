Rule options - Methods

withMessage (function(obj))

Defines the message to be returned when the rule fails to validate

rule.withMessage('Failed at {|}', function (obj) { return [new Date()]});

As shown in this example you can provide a method that will return an array of values to be interpolated in to the string in the provided order. This provided method receives the object being validated as an argument.

when (function(obj))

Allows you to specify a function returning true or false that indicates whether this rule should be run.

rule.when(function (obj) { return true; })

This provided method receives the object being validated as an argument.

validateCollection(true|false)

Forces the rule to run against each item in the array rather than against the array itself. If you supply no value it is the same as supplying true.
